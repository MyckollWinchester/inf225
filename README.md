## Instalación

Crear un entorno virtual y activarlo.

```
python -m venv .venv
```

```
.venv\Scripts\activate
```

Instalar dependencias.

```
pip install -r requirements.txt
```

## Ejecución

Activar el entorno virtual y ejecutar el programa.

```
.venv\Scripts\activate
```

```
py app.py
```
